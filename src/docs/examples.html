Examples

Basics

A timepicker in the simplest.

$('#ex_basic_1').intimidatetime({
	ranges: 1,
	buttons: [
			{ text: 'Done', action: function(inst){ inst.close(); } }
		],
	events: {
		refresh: function(e,inst){
			console.log('refreshed', inst.settings);
		},
		change: function(e,inst){
			console.log('changed value');
		}
	}
});

Add a few options, change the format.

$('#ex_basic_2').intimidatetime({
	format: 'MMM d, yyyy'
});
$('#ex_basic_2').intimidatetime('option','format');
